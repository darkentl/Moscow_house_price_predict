# Moscow FLat Price Prediction

Пользователь вводит основные параметры квартиры: площадь, количество комнат, этаж, станцию метро, время до метро и тип ремонта. Модель рассчитывает примерную стоимость квартиры и цену за квадратный метр.

## Датасет

В проекте используется датасет с объявлениями о продаже квартир в Москве:
https://www.kaggle.com/datasets/egorkainov/moscow-housing-price-dataset

## Признаки

В модели используются следующие признаки:

- `apartment_type` — тип квартиры
- `metro_station` — ближайшая станция метро
- `minutes_to_metro` — время до метро в минутах
- `rooms` — количество комнат
- `area` — общая площадь
- `living_area` — жилая площадь
- `kitchen_area` — площадь кухни
- `floor` — этаж
- `total_floors` — количество этажей в доме
- `renovation` — тип ремонта

Целевая переменная:

- `price` — цена квартиры

## Модели

В проекте сравниваются модели:

- Linear Regression
- ElasticNet
- Decision Tree Regressor
- Random Forest Regressor
- Extra Trees Regressor
- Gradient Boosting Regressor

Лучшая модель выбирается по минимальному значению метрики **MAE**.

## Метрики

Для оценки качества используются:

- `MAE` — средняя абсолютная ошибка
- `RMSE` — среднеквадратичная ошибка
- `R2` — коэффициент детерминации

## Сайт

Сайт позволяет пользователю ввести основные параметры квартиры и получить прогноз стоимости недвижимости в Москве.

### Возможности сайта

Пользователь может указать:

- тип квартиры;
- ближайшую станцию метро;
- время до метро в минутах;
- количество комнат;
- общую площадь квартиры;
- жилую площадь;
- площадь кухни;
- этаж;
- общее количество этажей в доме;
- тип ремонта.

Если пользователь не знает жилую площадь или площадь кухни, эти поля можно не заполнять. В таком случае сайт автоматически рассчитывает примерные значения на основе общей площади квартиры.

### Что показывает сайт

После нажатия кнопки расчёта сайт выводит:

- примерную стоимость квартиры;
- примерную цену за квадратный метр.
